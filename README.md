## 数据
经过编辑的波士顿房价数据集有490个数据点，每个点有三个特征。这个数据集编辑自加州大学欧文分校机器学习数据集库.

## 特征

RM: 住宅平均房间数量     
LSTAT: 区域中被认为是低收入阶层的比率      
PTRATIO: 镇上学生与教师数量比例     
## 目标变量

MEDV: 房屋的中值价格
